# 🌍 Travel Wallet Bot

Telegram-бот для отслеживания расходов во время путешествий с конвертацией валют.

## 📋 Описание

Travel Wallet — это мини-кошелёк для путешественника, который помогает:
- Создавать путешествия и отслеживать расходы
- Конвертировать валюты по актуальному курсу
- Видеть остаток в обеих валютах одновременно
- Вести историю расходов

## 🔧 Установка

1. Клонируйте репозиторий или скачайте файлы

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте конфигурацию:
```bash
cp .env.example .env
```

4. Отредактируйте файл `.env`:
```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
CURRENCY_API_KEY=your_currency_api_key_here
```

## 🚀 Запуск

```bash
python bot.py
```

## 📱 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Запустить бота и показать главное меню |
| `/newtrip` | Создать новое путешествие |
| `/switch` | Переключиться между путешествиями |
| `/balance` | Показать текущий баланс |
| `/history` | Показать историю расходов |
| `/setrate` | Изменить курс валют |

## 🎮 Использование

### Создание путешествия

1. Нажмите «Создать новое путешествие» или `/newtrip`
2. Введите название путешествия (например: «Китай 2026»)
3. Введите страну выезда (например: Россия)
4. Введите страну назначения (например: Китай)
5. Подтвердите или измените курс обмена
6. Введите начальную сумму в домашней валюте

Бот автоматически конвертирует сумму и создаст путешествие с двумя балансами.

### Добавление расходов

Просто отправьте число боту — бот посчитает это как расход в валюте страны пребывания.

Например, если активно путешествие «Россия → Китай»:
- Вы отправляете: `100`
- Бот считает: 100 CNY и конвертирует в рубли
- Бот предложит подтвердить расход

### Просмотр баланса

Нажмите «Баланс» или `/balance` — увидите остаток в обеих валютах.

## 🔌 API

Используется API [exchangerate.host](http://api.exchangerate.host) для получения актуальных курсов валют.

## 🗄️ База данных

Все данные хранятся локально в SQLite:
- Пользователи
- Путешествия (с валютными парами и балансами)
- История расходов
